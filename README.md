# ArrayList

## A CopyCat class of the original ArrayList<E> class

### Includes the following

1. Contructors that require no parameters or a set size
2. add methods
3. remove methods 
4. get / set methods
5. contains method
6. size method
7. isEmpty method
8. clear method
9. toString method
10. addAll method
11. indexOf / lastIndeOf methods
12. removeRange method
13. toArray method
14. equals method
